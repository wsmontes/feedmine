# Contributing to FeedMine

FeedMine is an open-source project maintained by Wawasoft. Contributions are
welcome through issues and pull requests.

## Code

1. Open an issue before starting a substantial feature or architectural change.
2. Keep pull requests focused and include tests when behaviour changes.
3. Do not add credentials, personal reading data, or copyrighted publisher
   content to the repository.

## Catalog

The OPML catalog is maintained separately at
[wawasoft/feedmine-repo](https://github.com/wawasoft/feedmine-repo). Use that
repository to propose source additions, corrections, removals, or category
changes. FeedMine downloads only feed metadata and published feed URLs; the
underlying articles, audio, and video remain the property of their publishers.
