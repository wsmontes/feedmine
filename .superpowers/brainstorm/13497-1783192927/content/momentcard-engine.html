Lógica de Seleção

1. Prioridade por contexto:
  🟢 Special date ativa → grupo [special] (8 templates)
  🟢 Night (21-5h) → grupo [noite] (8 templates)
  🟢 Dawn (5-8h) / Morning (8-12h) → grupo [abertura] (10 templates)
  🟢 Session > 20 min → grupo [leitura] (10 templates)
  🟢 Podcasts disponíveis → grupo [podcast] (8 templates)
  🟢 Streak > 3 dias → grupo [streak] (8 templates)
  🟢 Fallback → grupo [tom leve] (8 templates)

2. Preenche slots: avalia quais dados estão disponíveis, remove slots sem dados

3. Escore por completude: templates com mais slots preenchidos ganham prioridade

4. Randomização entre top 3: evita repetição. Com 60+ templates, combinações são enormes

5. Anti-repetição: nunca repete o mesmo template em menos de 2 horas

6. Refresh: reavalia a cada 30 segundos (já existe hoje). Muda quando o contexto mudar (hora vira, contagem sobe, etc.)
